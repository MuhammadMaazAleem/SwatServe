# Animations Directory

Place Lottie animation files here:

- `loading.json` - Loading animation
- `success.json` - Order success animation
- `empty_cart.json` - Empty cart animation

**Note:** The app works without animations. These are optional enhancements.

## Getting Lottie Files
- Visit: https://lottiefiles.com
- Download free animations
- Save as .json files here
