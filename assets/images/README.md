# Assets Directory

Place your image assets here:

- `logo.png` - App logo
- `splash.png` - Splash screen image
- `mountains.jpg` - Mountain/valley background
- Vendor images
- Product images

**Note:** The app works without these images. They're optional for enhancement.

## Recommended Image Sizes

- Logo: 512x512 px (PNG with transparency)
- Splash: 1080x1920 px (PNG)
- Background: 1920x1080 px (JPG)
- Vendor images: 800x600 px
- Product images: 600x600 px
